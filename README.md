# Micromouse
UCLA IEEE Micromouse project (2025-2026). Developed with teammates James Tao and Aidan Pan. Base template provided by project leads. PID and floodfill algorithms independently implemented.

## Hardware
- STM32F205 microcontroller
- [Schematics & PCB layout](https://andraew.github.io/)

## Software
- PID controller to move straight and turn
- Floodfill maze-solving algorithm

## Setup
1. Open in STM32CubeIDE
2. Build and flash to board
3. Tune PID and IR values
